Reactive Audio/Visual

Webcam+Drum machine

In this project I explored machine learning using a SimpleWebcam input and a drum machine output called Chuck beats. By syncing these systems together using the program Wekinator, I was able to create an interactive live visual/audio experience. The video below is a demo of the recording and training process.

Wave Spectrums

As a lover of music and light shows at concerts, I am interested in the ways we can use machine learning to create meaningful and aesthetically pleasing visuals. These two videos are an extension of my first project and I would want to combine them together to create an interactive, colorful experience. I enjoyed playing around with the Wave Spectrum settings in Adobe AfterEffects to make this audio-reactive glitch and I will definitely be experimenting with this more in the future.
